MYSHINE WEBSITE BESTANDEN

Upload deze bestanden naar je GitHub repo en vervang je oude:
- index.html
- style.css
- script.js

Belangrijk:
- Zorg dat je afbeelding exact 'hero.png' heet
- Zet hero.png in dezelfde hoofdmap als index.html

Structuur:
/
index.html
style.css
script.js
hero.png

Wat deze versie doet:
- Volledig responsive
- Mobiele versie inbegrepen
- 9:16-vriendelijk
- Detecteert mobiel/desktop via JavaScript + viewport
- Mobiel menu
- Contactgegevens en socials zichtbaar
- Boekingsformulier met controle dat enkel zondag/maandag gekozen kan worden

Wat nog NIET automatisch werkt:
- echte mailverzending

Daarvoor moet je later nog EmailJS of Formspree koppelen met jouw accountgegevens.
